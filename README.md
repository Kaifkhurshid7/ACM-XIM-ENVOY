# ACM Media Platform

A full-stack media platform for ACM student chapters.

## Project Structure

- `backend/`: Express.js server with Node.js and Mongoose.
- `frontend/`: React.js frontend with Vite and React Router.

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. Create a `.env` file with `PORT` and `MONGO_URI`.
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Features
- User Authentication (JWT)
- Post Creation and Interaction
- Forum Discussions
- Event Management
- Admin Control Panel
